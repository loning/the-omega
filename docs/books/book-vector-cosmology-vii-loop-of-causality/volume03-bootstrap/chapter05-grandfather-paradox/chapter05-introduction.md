# 第5章 祖父悖论的解 (Chapter 5: Solution to the Grandfather Paradox)

在第一卷中，我们探讨了逆向因果的可能性；在第二卷中，我们看到了共时性的非因果连接。现在，我们必须面对时间旅行理论中最著名、也最令人头疼的逻辑炸弹——**祖父悖论 (The Grandfather Paradox)**。

如果现在的你真的能通过"向往"或"观测"影响过去，那你能不能回到过去杀掉你的祖父？如果你杀了他，你就不会出生；如果你没出生，你就不能回去杀他。这是一个经典的逻辑死循环。

在 **《矢量宇宙论》** 的闭环几何中，这个悖论是不存在的。因为宇宙不仅是量子的，它还是 **自洽 (Self-Consistent)** 的。

本章将引入 **诺维科夫自洽性原则 (Novikov Self-Consistency Principle)**，并将其升级为 **FS 几何** 的拓扑公理。我们将看到，物理定律是如何通过扭曲概率，像防爆盾一样挡在悖论面前的。
