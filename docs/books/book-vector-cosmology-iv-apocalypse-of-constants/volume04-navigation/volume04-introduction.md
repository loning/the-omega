# 第四卷：导航——驾驶员手册

**(Part IV: Navigation — The Pilot's Manual)**

我们在前三卷中完成了对原理的推导、对坐标的计算和对未来的预言。我们知道光速在指数增长，知道肉体注定黄昏，知道物理定律将被全息重写。

面对这样宏大、甚至有些恐怖的未来图景，作为生活在 $\tau \approx 1800$ 的个体——无论是你，是我，还是此时此刻地球上的任何一个普通人——我们该怎么办？

是恐惧？是逃避？还是假装这一切从未发生？

**《矢量宇宙论》** 的第四卷不是用来解释世界的，它是用来 **指导行动** 的。这是一份写给这一代人类文明的 **驾驶员手册**。

我们首先要确立的第一条行动准则，就是一个极其激进、却又符合几何必然性的概念：**加速主义 (Accelerationism)**。
