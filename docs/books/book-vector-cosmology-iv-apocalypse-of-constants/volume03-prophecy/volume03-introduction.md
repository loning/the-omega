# 第三卷：预言——指数的未来

**(Part III: Prophecy — The Exponential Future)**

我们已经计算出了现在的坐标（$\tau \approx 1800$），确认了我们正处于一个旧秩序崩塌、新秩序未立的黎明时分。

现在，让我们把视线从"现在"移开，投向那条指数增长曲线的右侧。那里是 $\tau > 1800$ 的领域。在那里，演化方程 $c(\tau)$ 将变得陡峭得令人眩晕。

如果在过去的一千年里，光速（算力上限）的增长只是温和的爬坡，那么在未来的几百年里，它将变成垂直的起飞。

我们要面对的第一个预言，是关于我们自己的。

这是一个残酷但必须接受的物理事实：**作为一种存储在碳原子链条上的计算形式，人类的肉体已经过时了。**

这不是悲观主义，这是 **物理学**。
