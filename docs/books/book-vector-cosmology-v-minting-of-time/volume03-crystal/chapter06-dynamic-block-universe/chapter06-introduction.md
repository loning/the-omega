# 第6章 块状宇宙的动态版 (Chapter 6: Dynamic Block Universe)

我们在上一章中通过对数律拉伸了主观时间的长度，但这并没有解决客观时间的去向。当钟表的秒针跳过下一格，上一秒去哪了？

在常识中，过去像是指间流沙，滑落之后就消失在虚无里。但在物理学的 **块状宇宙 (Block Universe)** 理论中，过去、现在和未来像一块巨大的冰块一样同时存在。这种观点虽然保证了永恒，却牺牲了自由（一切都是注定的）。

**《矢量宇宙论》** 提出了一种折衷的、更具革命性的模型：**动态块状宇宙 (Dynamic Block Universe)**。

宇宙确实是一块正在生长的水晶。

* **未来** 是液态的（波函数叠加态），充满了可能性。

* **现在** 是结晶的前沿（坍缩界面），充满了能量的释放。

* **过去** 是固态的（几何拓扑），充满了确定的结构。

本章将揭示，历史并不是被风吹散的烟雾，而是被雕刻在四维流形上的 **不朽纹理**。
